# Média-Queries

Les `media-queries`permettent d'écrire des règles de css en fonction de la taille ou du type de l'écran. Ces règles s'écrivent avec le mot-clé `@media`

Par exemple : 

```css
@media screen and (max-width: 640px) {
  .bloc {
    display:block;
    clear:both;
  }
}
```
